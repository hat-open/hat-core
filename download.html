Python packages

Python packages require Python 3.8 or newer and can be installed with pip. 
$ pip install <package>
where <package> is one of:
  • hat-util
  • hat-aio
  • hat-json
  • hat-qt
  • hat-peg
  • hat-stc
  • hat-sbs
  • hat-chatter
  • hat-juggler
  • hat-duktape
  • hat-sqlite3
  • hat-asn1
  • hat-drivers
  • hat-syslog
  • hat-orchestrator
  • hat-monitor
  • hat-event
  • hat-gateway
  • hat-gui

JavaScript packages

JavaScript packages can be installed with npm. 
$ npm install <package>
where <package> is one of:
  • @hat-core/util
  • @hat-core/renderer
  • @hat-core/future
  • @hat-core/juggler
